# Code of Conduct

`lintforge` aims to be a welcoming, practical project for people building better
Dart and Flutter tooling.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent, especially when reviewing issues and pull requests.
- Keep feedback specific to the work.
- Make room for contributors with different experience levels.

## Unacceptable Behavior

Harassment, personal attacks, discriminatory language, intimidation, and
sustained disruptive behavior are not acceptable in this project.

## Enforcement

Maintainers may edit or remove comments, close issues, reject pull requests, or
block participants when behavior harms the project or its contributors.

If you need to report a concern, contact the maintainers through the most
private GitHub channel available for this repository. If a public issue is the
only available channel, include only non-sensitive details and ask for a private
follow-up path.
